# Form-a-word

## Free Fire-style teams setup

This project defines 4 Free Fire-style squads, each with 6 members, in `teams.py`.

Project link: https://github.com/sentient-dev/Form-a-word

Run app: [Open teams.py](./teams.py)

Usage:

```bash
python3 teams.py
```

- Phoenix Brigade: 6 members
- Shadow Battalion: 6 members
- Royale Hunters: 6 members
- Crimson Squadron: 6 members

Run with:

```bash
python3 teams.py
```

The script now outputs each squad and its squadmates, using Free Fire-like naming.

## Free Fire-style simulation

`teams.py` now includes a simple battle royale simulator:

- Random elimination rounds between squads
- Prints round events
- Declares last squad standing as winner
- Supports up to 30,000 levels in simulation (default)

Usage is still `python3 teams.py`.
